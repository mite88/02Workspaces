쌍다옴표와 홑따옴표

JAVA에서는 문자열을 표현할때 "문자열" 형태로 사용하고
문자를 표현할때는 '문자' 형태로 사용한다.
참고 : '는 IE(인터넷익스플로러)에서 사용이 불가능하다.

문자포맷을 그대로 웹브라우저에 표현

public class HelloWorld {

	public static void main(String[] args) {

		/*
		 * 2022.03.21 
		 * System.out.println : 줄단위글자 출력 
		 * 실행은 ctrl+F11
		 */
		System.out.println("안녕하세요. Java입니다.");
		System.out.println("자동완성은 Ctrl+space입니다.");

		System.out.println("테스트 파일임");
		System.out.println("JRE-11로 설정함");
	}

}
